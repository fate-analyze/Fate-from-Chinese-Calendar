# Sxwnl-cpp  
寿星天文历(万年历)c++版  
Sxwnl for cpp version
# Build
## 1.Use cmake  
```bash
# build or update bin
./autobuild.sh

# clean obj files and build
./rebuild.sh

```
## 2.Use make  
```bash
# clean obj files
make clean
# for windows
make cleanw

### Build test0
make test0
### Build test1
make test1
### build all
make

```

# Run test  
```bash
# cmake build
./build/test0
./build/test1
# make build
./test0
./test1
```
