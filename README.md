# sxwnl-cpp  
寿星天文历(万年历)c++版  
sxwnl for cpp version
# build
## 1.use cmake  
```bash
# build or update bin
./autobuild.sh

# clean obj files and build
./rebuild.sh

```
## 2.use make  
```bash
# clean obj files
make clean
# for windows
make cleanw

# build test0
make test0
# build test1
make test1
# build all
make

```

# run test  
```bash
# cmake build
./build/test0
./build/test1
# make build
./test0
./test1
```
