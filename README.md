# sxwnl-cpp  
寿星万年历c++版
sxwnl for cpp version
# build
## 1.use cmake  
```bash
bash autobuild.sh
```
## 2.use make  
```bash
# clean obj files
make clean
# for windows
make cleanw

# build test0
make test0
# build test1
make test1
# build all
make

```

# run test  
```bash
# cmake build
./build/test0
./build/test1
# make build
./test0
./test1
```
